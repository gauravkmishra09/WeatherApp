# 🌦️ Weather App

A simple Weather Application that displays real-time weather information for any city.

## 🚀 Features
- Search weather by city name
- Displays temperature, humidity, and weather condition
- Responsive UI
- Real-time weather data using API

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Weather API
